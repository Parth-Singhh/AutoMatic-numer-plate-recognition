# 🚘 Automatic Number Plate Recognition (ANPR) System

A modern AI-powered Automatic Number Plate Recognition (ANPR) system built using Python, OpenCV, OCR, and Streamlit.

---

# 📌 Project Overview

This project detects and recognizes vehicle number plates from uploaded images using Computer Vision and Optical Character Recognition (OCR).

The system:

* detects the number plate region,
* extracts the plate,
* recognizes text using OCR,
* and displays the result in a modern web interface.

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone <your-repository-link>
```

---

## 2️⃣ Open Project Folder

```bash
cd AutoMatic-numer-plate-recognition
```

---

## 3️⃣ Create Virtual Environment

```bash
python -m venv .venv
```

---

## 4️⃣ Activate Virtual Environment

### Windows PowerShell

```powershell
.\.venv\Scripts\Activate.ps1
```

---

## 5️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run the Application

## Streamlit Web App

```bash
python -m streamlit run app.py
```
